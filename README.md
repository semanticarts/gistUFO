# SemanticArtsSubGistTemplate

A template repository for creating new subGist ontologies at Semantic Arts.

## Goals

The goal of this template is to provide a standard starting point for us to create new subGists for emerging domain work. This repository is based on patterns from a number of previous projects and is mostly barebones to allow for customization after the template is used.

More information about this template and associated resources can be found at the following link with its child pages: [2025 Ontology Tooling Development Project](https://semarts.atlassian.net/wiki/spaces/TRR/pages/2860417039/2025+Ontology+Tooling+Development+Project)

## Structure

Each sub-directory is populated with a Markdown file that one can modify as they see fit after the template has been used.

## How-to

To use this template, there is a button on GitHub to "Use this template". From here, one has dialog options to create a new repository based on the repo **as it exists when the new repository is created**, meaning any commit history and commits *after* the new repository is created will not be included. One has the option to include all branches as well, which may be useful if we store variations of this template in other branches (TBD).
