# gistUFO

This repository stores artifacts related to the alignment of [gist](github.com/semanticarts/gist) with the [Unified Foundational Ontology](https://ontouml.org/ufo/) (UFO) and [OntoUML](https://ontouml.org/ontouml/) via mapping to [gUFO](https://nemo-ufes.github.io/gufo/), a lightweight OWL implementation of UFO. This project has recently developed into a collaboration between Semantic Arts and the gUFO team. 

The ontologies directory contains work-in-progress mappings from gist to gUFO Types and Individuals. The queries directory contains SPARQL queries to be used for ontology/mapping validation and inference. The standard gist toolset is in the tools directory, along with a Python script to be used with a subset of validation queries.
